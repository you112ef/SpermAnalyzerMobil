## 🚀 كيفية الاستخدام

### 1. تحميل الأرشيف
```bash
# الملف متوفر في المجلد الجذر
sperm-analyzer-android-app.tar.gz
```

### 2. استخراج الملفات
```bash
tar -xzf sperm-analyzer-android-app.tar.gz
cd android-app
```

### 3. فتح في Android Studio
1. افتح Android Studio
2. اختر "Open an Existing Project"
3. حدد مجلد `android-app`
4. انتظر مزامنة Gradle

### 4. إضافة نماذج الذكاء الاصطناعي
**مهم جداً**: التطبيق يحتاج نماذج مدربة للعمل

ضع الملفات التالية في `app/src/main/assets/`:
- `yolov8_sperm.tflite` (نموذج الكشف الرئيسي)
- `deepsort_features.tflite` (نموذج التتبع - اختياري)

### 5. بناء التطبيق
```bash
# للاختبار
./gradlew assembleDebug

# للإنتاج
./gradlew assembleRelease
```

## 🧠 تدريب النماذج

### البيانات المطلوبة
- صور مجهرية للحيوانات المنوية (تكبير 400x)
- تسميات بتنسيق YOLO مع فئات:
  - 0: حيوانات منوية تقدمية
  - 1: حيوانات منوية غير تقدمية  
  - 2: حيوانات منوية ثابتة

### عملية التدريب
```python
from ultralytics import YOLO

# تحميل النموذج
model = YOLO('yolov8n.pt')

# التدريب
results = model.train(
    data='sperm_dataset.yaml',
    epochs=100,
    imgsz=640,
    batch=16
)

# تصدير إلى TensorFlow Lite
model.export(format='tflite', int8=True)
```
